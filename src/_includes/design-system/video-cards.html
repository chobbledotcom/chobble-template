{%- comment -%} Video cards grid - displays YouTube videos as lazy-loaded thumbnail cards. Uses thumbnail placeholders instead of iframes for zero-bandwidth initial load. The iframe is only loaded when the user clicks/taps (progressive enhancement). Without JavaScript, clicking opens YouTube in a new tab. Parameters: - videos: Array of video objects with properties: - id: YouTube video ID (required) - title: Video title (required) - reveal: If true, adds data-reveal animation (default: true) - heading_level: Heading level for titles, defaults to 3 (h3) Usage: {% include "design-system/video-cards.html", videos: page.videos %} Where page.videos = [ { id: "dQw4w9WgXcQ", title: "Video Title" }, ... ] {%- endcomment -%} {%- assign reveal = reveal | default: true -%} {%- assign heading_level = heading_level | default: 3 -%}
    {%- for video in videos -%}
    {%- youtubeThumbnail video.id, video.title -%}
    {{ video.title }} {%- endfor -%}
